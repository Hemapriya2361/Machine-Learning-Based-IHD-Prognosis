# Cardiovascular Disease Temporal Prognosis App

An AI-powered full-stack web application for cardiovascular disease risk stratification and survival analysis.

## Features
* **Machine Learning Backend**: Utilizes Random Survival Forests (`rsf_bb.pkl`) for clinical temporal prognosis.
* **Interactive Frontend**: Built with React for sleek user inputs and dynamic risk data visualization.
* **Flask API**: Lightweight communication layer between the React UI and the ML model.

## Tech Stack
* **Frontend**: React, JavaScript, HTML5, CSS3
* **Backend**: Flask, Python 3.12, Joblib
* **ML Libraries**: Scikit-Learn, Scikit-Survival
